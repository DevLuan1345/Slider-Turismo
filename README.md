## 📌 Projeto Slider Turismo

O Slider Turismo é uma aplicação web interativa que apresenta destinos turísticos de forma visual e dinâmica, utilizando um sistema de carrossel (slider) com transições suaves e foco em experiência do usuário. O projeto destaca cidades famosas como Bali, Paris, Roma e Veneza, trazendo descrições atrativas e imagens imersivas.

## 🎯 Objetivo

Desenvolver uma interface moderna e responsiva que simula um slider de turismo, permitindo ao usuário explorar diferentes destinos de maneira intuitiva, com foco em design, animações e usabilidade.

## ⚙️ Funcionalidades

- 🎞️ Slider interativo com navegação por botões (anterior e próximo)
- 🖼️ Exibição de imagens em destaque com efeito de profundidade
- 📝 Conteúdo descritivo dinâmico para cada destino
- 🎯 Miniaturas (thumbnails) para navegação rápida
- ✨ Animações suaves na transição entre slides
- 📱 Layout adaptado para diferentes tamanhos de tela (responsivo)

## 🧱 Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨

## 📱 Responsividade

O projeto foi desenvolvido para funcionar em diferentes dispositivos, adaptando:

- 📲 Layout para telas menores (mobile)
- 📏 Redimensionamento de textos e elementos
- 🔄 Ajuste automático do conteúdo para melhor leitura
- 🎯 Organização dos elementos para manter a usabilidade

## 🎨 Destaques do Design

- Interface inspirada em aplicações modernas
- Uso de imagens em tela cheia com overlay escuro
- Tipografia limpa e legível
- Botões com efeitos visuais e interativos
- Navegação simples e intuitiva
